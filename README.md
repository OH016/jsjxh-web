## 本地搭建

构建要求:

- Node.JS `18.18.0`
- npm `9.8.1`

在根目录中使用`npm i`解决依赖关系

```shell
npm i
```

在根目录中使用`npm start`启动服务器

```shell
npm start
```
进行打包

```shell
npm run build
```
